# Spider-Man Portfolio — Project Instructions

## Project Context
This is a **Next.js + TypeScript** Spider-Man themed developer portfolio with animated components, GSAP animations, and Tailwind CSS.

- `web-performance-optimization` — Core Web Vitals, lazy loading, bundle size
- `seo-fundamentals` — meta tags, semantic HTML, structured data
- `wcag-audit-patterns` — accessibility compliance
- `vercel-deployment` — deployment and preview URLs

## Project Rules
- Use Next.js App Router (not Pages Router)
- All components are in `src/components/`
- Use TypeScript strict mode
- Animations should be smooth (GSAP or Framer Motion)
- Mobile-first responsive design
- Spider-Man theme: red/blue/black color palette with web motifs
